# 🏀 NBA Playoffs 2026 — Prediction League

## פרסום ב-5 דקות עם Vercel (ללא ניסיון קודי)

---

### שלב 1 — צור חשבון GitHub (אם אין לך)
1. היכנס ל-**github.com**
2. לחץ **Sign up** → הירשם עם אימייל

---

### שלב 2 — העלה את הפרויקט ל-GitHub

1. היכנס ל-**github.com/new** (צור repository חדש)
2. תן שם: `nba-playoff-league`
3. השאר **Public** (נדרש לפלאן חינם של Vercel)
4. לחץ **Create repository**
5. בדף שנפתח, לחץ על **"uploading an existing file"**
6. גרור את כל התיקייה `nba-league` (את כל הקבצים שקיבלת) לדף
7. לחץ **Commit changes**

---

### שלב 3 — פרסם ב-Vercel

1. היכנס ל-**vercel.com**
2. לחץ **Sign up with GitHub** (כניסה עם חשבון GitHub)
3. לחץ **Add New Project**
4. בחר את ה-repository `nba-playoff-league`
5. Vercel יזהה אוטומטית Vite — לחץ **Deploy**
6. תוך ~1 דקה תקבל קישור כמו:
   ```
   https://nba-playoff-league.vercel.app
   ```

**שלח את הקישור לחברים — זהו!** 🎉

---

### שלב 4 — הגדר אדמין (חשוב!)

1. פתח את האפליקציה בדפדפן
2. התחבר עם:
   - **Email:** `admin`
   - **Password:** `nba2026admin`
3. היכנס ל-**Settings → Change Admin Password**
4. הגדר סיסמה חדשה שרק אתה יודע!

### מה עושה כל תיקייה/קובץ

```
nba-league/
├── src/
│   ├── App.jsx      ← כל קוד האפליקציה
│   └── main.jsx     ← נקודת כניסה (לא לגעת)
├── index.html       ← עמוד הבסיס (לא לגעת)
├── package.json     ← תלויות הפרויקט (לא לגעת)
├── vite.config.js   ← הגדרות build (לא לגעת)
└── vercel.json      ← הגדרות Vercel (לא לגעת)
```

### שאלות נפוצות

**Q: האפליקציה חינמית לחלוטין?**
A: כן. GitHub ו-Vercel חינמיים לפרויקטים כאלה.

**Q: האם הנתונים נשמרים?**
A: הנתונים נשמרים ב-localStorage של כל דפדפן בנפרד. המשמעות: כל משתמש רואה את הנתונים שלו על המכשיר שלו. אם תרצה בסיס נתונים משותף אמיתי — זה שלב הבא.

**Q: האם הלוגואים יעבדו?**
A: כן! ברגע שהאפליקציה רצה על Vercel (לא בתוך Claude), לוגואי ESPN נטענים מיד.

**Q: איך מעדכן אחרי שינוי קוד?**
A: פשוט העלה את App.jsx המעודכן ל-GitHub → Vercel מתעדכן אוטומטית תוך דקה.

---

### ⚠️ הערה חשובה על נתונים

האפליקציה שומרת נתונים ב-`localStorage` — כלומר **כל משתמש על המכשיר שלו**. זה מתאים לשימוש אישי. לניהול ניחושים של קבוצה גדולה (שכולם רואים אחד את השני בזמן אמת), יש צורך בבסיס נתונים כמו Supabase — ניתן להוסיף זאת בעתיד.
